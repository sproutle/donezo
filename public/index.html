About this application: This was a practice assignment to learn about programming. This site uses Jam, Node.js, Bootstrap.css, CouchDB with Nano (the node library for couch), Express, Dombuilder, JQuery plugins, and Backbone with Underscore. Credits go to Scott Sanders (project leader), Lauren Caswell (intern), Jonathan Luna (intern), and Tim Caswell (teacher). Named Donezo, this site is a todo app that is in the form of many projects with many tasks per project. The user can add whichever projects they want, existing or new, to their account and then start working. The unique thing about this application is that more than one user can be included in the same project and everyone included in the project can edit it or comment it at the bottom. To get to your project page, click on your username in the top-right corner (after logged in) to begin exploring how interactive this todo app is with other users.
